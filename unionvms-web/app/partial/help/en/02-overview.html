Main navigation bar

  • Once logged in, the main navigation bar becomes visible. This navigation bar is consistent true system and will be shown at all times.
  • You navigate true the system by clicking on the links in the navigation bar to reach desired page.

Date and time

  • Date and time is shown in top of the page.

Active Alerts Icon

  • An Icon showing how many alerts is currently active.
  • By clicking on the icon, you automatically link to the alerts page where you find extended information regarding the alerts.
  • More information about the Alerts, see Section 7: Alerts.

Language Setting

  • A flag icon is illustrating the language of your choice.
  • To change language settings, click on the flag icon and choose preferred language in the dropdown list by clicking at it.

User Settings

All personal user settings are listed in a dropdown list under the username found up in the right corner of the page. For more information, see Section 10: User.

  • Click on the arrow next to the user name to reach a drop list with four different choices.
Update contact details
  • Select the option "Update contact details" in the dropdown list.
  • Fill in your new contact details in the pop-up window and the click on the "Save" button.
Change password
  • Select the option "Change password" in the dropdown list.
  • Fill in the obligational information in the pop-up window and then click on the "Save" button.
  • The new password will now be saved.
Change security answer
  • Select the option "Security answer" in the dropdown list.
  • Fill in all obligational information in the pop-up window and then click on the “Save” button.
Sign out
  • Click on the "Sign out" button in the dropdown list.
  • Sign out from the system and end up on the start page again.
