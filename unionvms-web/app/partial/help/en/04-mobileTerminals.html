The following notification icon appears when a mobile terminal has been created but not assigned to a carrier. For more information about how to link a terminal to assert, see “Assign mobile terminal to assert” in this Section.

Detailed information
  • To reach more detailed information about a terminal, click on the “Detail” button to the right in the list. See illustration above.
Status

The status of a mobile terminal can be Active or Inactive. When a mobile terminal has been made inactive, it will be greyed out in the list.

Select asset group

By selecting a group, all terminals assigned to the carriers in that group get listed.

Search for mobile terminals

Use the search fields to find mobile terminals.

  • Fill in the different fields with information of choice.
  • Click on the “Search” button to find matching mobile terminals.
  • The search result will be listed below with all relevant information.

Add new mobile terminal

When adding a new terminal, click on the "Add new" button to reach the "New mobile terminal" page.

Please follow the following instruction steps in order to add a new mobile terminal to the system.

It is always possible to cancel and go back to the previous screen. Click on the close icon. See illustration in picture below.

1. Select transponder system

First select which transponder system to be used. The list of available systems will be shown as a dropdown menu when clicking on the icon. See picture below.

Please note that the amount of information fields will vary depending on selected transponder system.

2. Ocean region

Select occean region from four available options in the dropdown list. It is possible to select multiple alternatives.

3. Insert relevant information

4. Add communication channel. At least one communication channel must be added in order to complete the form. It is possible to add diffirent channels.

The transponder systems AIS and IDRIUM do not have this requirement.

It is possile to reorder communication channels. The first added one will be used as defualt for polling.

Assign mobile terminal to asset

  • Use the search field to find a mobile terminal or select the mobile terminal to link by clicking on the “Details” button in the right column in the list.
  • Click on “Link asset” to open search field for assets. See pictures below.
  • Use the search fields to find asset and then click on the search button. See picture below. If no asset is found, a new one can be created and added to the local database.
  • Choose asset by clicking on the “Select” button. See picture below.
  • Click on the ”Link” button to connect the asset to the mobile terminal. See picture below.
  • Type in reason for changesa and click on the ”Link” button.
  • A feedback message will show for a few seconds, confirming the action.

Unlink asset from mobile terminal

  • To unlink a mobile terminal from an asset, select the mobile terminal to link by clicking on the “Details” button in the right column in the list.
  • Click on the “Unlink” button.
  • Type in reason for changes and click on the “Unlinking” button to save changes.
  • A feedback message will show for a few seconds, confirming the action.

Access mobile terminal history

  • Select mobile terminal true the “View Details” button.
  • The mobile terminal’s details will open. In the lower left corner is the “View history” link, click in it to open the history information.
  • To close the history window and go back to previous view, click on the “Close” button.

Activate mobile terminal

  • Select mobile terminal true the “View Details” button.
  • Click on the “Activate” button, up in the right corner.
  • Type in reason for change and click on the “Activate” button.
  • A feedback message will show for a few seconds, confirming the action.

Inactivate mobile terminal

  • Select mobile terminal true the “View Details” button.
  • Click on the “Inactive” button, up in the right corner.
  • Type in reason for changes and click on the “Inactivate” button.
  • A feedback message will show for a few seconds, confirming the action.
