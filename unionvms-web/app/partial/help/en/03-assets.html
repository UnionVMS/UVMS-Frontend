Section 3 Assets

The Assets view provides a list of all existing assets. This is where to add new assets to the system.

Search for carriers

  • Search by using a single keyword or keyword combinations to find all matching results presented in a list.
  • Search can also be done as a wildcard. Type a letter in the desired search word and add a ’*’ next to it in order to do a search for all matches with the selected letter in it. See example below:
Advanced Search

Expanded search area to reveal advanced search options.

  • For more search options, click on the link “Advanced search”. See pictures below.
  • To speed up the search functionality, perform search through specific fields only.
Sort carriers
  • The result can be sorted in descending order, click on the arrow in preferred column of the result list to change the order of the presented result. See picture below.
  • Click again on the arrow and it sorts the list in an ascending order.
View details
  • To view extensive details about the asset, click on the “Details” button under the column “Details” in the result list, See picture below.
  • To go back to the previous page, click on the “Close” button.

Edit Selection

Once search selections have been made and the result has come up, it is possible to edit the selection by choosing from three different alternatives in the “Actions” dropdown list;

  • Save as group
  • View on map
  • Export selection
Save as group

By selecting certain results of the search, it can be saved into a group.

  • Select which results should be saved by clicking in the box left to it. See picture below.
  • Once the selection/ selections have been made, choose the alternative “Save as a group” in the dropdown list under “Actions”. See picture below.
  • Add a title to name the group or choose to save as an already existing group by select an alternative in the dropdown list. See picture below.
View on map

By selection this option, the result will be shown on a map.

  • Select which asset or assets you want to view on a map by clicking in the box left to it. See picture below.
  • Once the selection/ selections have been made, choose the alternative “View on map” on in the dropdown list under “Actions”. See picture below.
  • To close map view and go back to search result, click on the cross button placed in the upper right corner. See picture below.
Export selection

It is possible to export one or multiple mobile terminals from the list to an Excel file.

  • Select required mobile terminals by clicking in the box left to it. See picture below.
  • Once the selections have been made, choose the alternative “Export selection” in the dropdown list under “Actions”. See picture below.
  • The file will be downloaded to the computer and stored in “Downloaded files”.
Add new asset
  • Click in the “New Asset” button.
  • Fill in all needed details in the form and click on the “Save asset” button.

Note that all required fields must be correct filled in. Otherwise a feedback message will appear saying that the asset cannot be created. See picture below.

  • Once the asset is added, a feedback message will confirm that and link back to the main asset page.
Update Asset

It is possible to update information if the asset has an international source.

  • To update an asset, click on the “Details” button to open up the asset settings.
  • Click on the “Update” button up to the right, see picture below
  • A feedback message will show for a few seconds, confirming that the asset has been updated

Archive asset
  • To archive an asset, click on the “Details” button to open up the asset settings.
  • Click on the “Archive” button up to the right, see picture below.
  • Then click “Yes” in the pop-up window.
  • A feedback message will show for a few seconds, confirming that the asset has been archived.
